Data Visualization and Download

This page will provide access to historical data that are collected for the Freedom Field system.  Data are primarily collected in a server within the Freedom Field using the hardware and software developed by General Electric (GE).  An arrangement has been made to upload the collected data to a NIU server (located within the Department of Technology).  The data are then converted to an appropriate file format and saved to a secured server.  The whole transfer and conversion process are implemented using an automated system without any human intervention.  A number of Web applications are then developed to visualize the data in graphical form.  A user can choose the duration of displayed data.

In addition to the graphical visualization a user can download the data to his/her local drive and analyze them as necessary. Access to the data is password protected.  If you like to an access, please contact with Dr. Azad (aazad@niu.edu) for an account.  If you already have an account please click on the button below.
